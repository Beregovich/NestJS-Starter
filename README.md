## TypeORM migrations

```bash
#create empty migration 
$ typeorm migration:create ./src/migration/migration-name
#generate
$ yarn migration:generate ./src/migration/migration-name  

#run
$ yarn migration:run 

#revert
$ yarn mmigration:revert
```
